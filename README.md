A minimal Node.js project for user registration and login, including basic authentication and API endpoints.

## Installation

Create a .env file
```
PORT=3000
MONGO_URI=mongodb://localhost:27017/register-login
JWT_SECRET=your_secret
```
Start the server:
```
npm start
```

>The frontend part of this project was created with the help of AI.
-----
Features
- User registration & login
- Password hashing
- JWT & Cookie-based authentication
